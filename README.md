## Proyecto

### Proyecto 1 - Casa Inteligente con Control de Ambientes y Ventilador Automatizado

Este proyecto consiste en el diseño e implementación de una maqueta funcional de una casa inteligente, en la cual se integran múltiples componentes electrónicos y principios de lógica combinacional y secuencial para simular un sistema domótico.

El sistema permite el control de distintos ambientes (sala, comedor, cocina, baño y habitación) mediante LEDs que representan la iluminación, así como la integración de un ventilador simulado con un motor DC y una puerta automatizada controlado por un servomotor. Todo el sistema es gestionado por un microcontrolador Arduino, el cual coordina las entradas y salidas digitales.

Además, se incorpora comunicación inalámbrica mediante un módulo Bluetooth, lo que permite controlar el sistema de forma remota desde un dispositivo móviil. El estado del sistema se muestra en tiempo real a través de una pantalla LCD y LEDs indicadores, brindando retroalimentación visual al susuario.
