## Prácticas Desarrolladas

### Práctica 1 – Visualizador de 7 Segmentos

Implementación de un sistema de visualización bidireccional utilizando lógica combinacional, displays de 7 segmentos y compuertas transistorizadas.

### Práctica 2 – LogicCalc (ALU Básica)

Diseño e implementación de una calculadora basada en lógica combinacional capaz de realizar operaciones aritméticas y lógicas mediante una Unidad Aritmética Lógica (ALU).

### Práctica 3 – Carrusel Automatizado con Control de Acceso Seguro

Implementación de un sistema digital que integra autenticación mediante contraseña, control de errores, manejo de un motor DC con puente H y temporización de funcionamiento utilizando lógica combinacional y secuencial.
